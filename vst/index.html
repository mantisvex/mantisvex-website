MASS

An adaptive harmonic density processor. MASS adds weight and harmonic density with material-flavoured saturation — Steel, Velvet, Concrete, Smoke, Obsidian, Gold — plus mid/side processing, parallel mix, width and output control, A/B compare and a built-in spectrum display.

DOOM COMPRESSOR

A recreation of Mick Gordon's infamous DOOM (2016) upward compressor. Instead of taming peaks it drags everything — reverb tails, noise floor, artifacts — up to 0 dBFS, turning dull patches into dense, distorted walls of doom. Floor, Ceiling, Drive and Crush controls with a live LIFT meter.

// INSTALL

  1. Download the plugin from its DOWNLOAD button above (unzip if needed).
  2. Copy the .vst3 into your VST3 folder (e.g. C:\Users\<you>\VST3 or %COMMONPROGRAMFILES%\VST3).
  3. Rescan plugins in your DAW. Load. Corrupt the signal.

The binaries are unsigned, so Windows SmartScreen may warn on first run — click More info → Run anyway, or right-click the file → Properties → Unblock. Everything is open source under MIT — read or build the code yourself on GitHub.
