MASS

An adaptive harmonic density processor. MASS adds weight and harmonic density with material-flavoured saturation — Steel, Velvet, Concrete, Smoke, Obsidian, Gold — plus mid/side processing, parallel mix, width and output control, A/B compare and a built-in spectrum display.

// INSTALL — MASS

  1. Download MASS.vst3 from the latest release.
  2. Copy it into your VST3 folder (e.g. C:\Users\<you>\VST3).
  3. Rescan plugins in your DAW. Load. Add weight.

The binary is unsigned, so Windows SmartScreen may warn on first run — click More info → Run anyway, or right-click the file → Properties → Unblock. It's open source under MIT: read or build the code yourself on GitHub.
